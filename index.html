Keyboard shortcuts

  1. ⌘+C and ⌘+V: copy and paste
  2. ⌘+⎇+G on preview: go to page
  3. ⌘+F: find
  4. ⌘+R on google: refresh page
  5. ⌘+⇧+3: screenshot; ⌘+⇧+4: screenshot a portion of the screen
  6. ⌘+⇧+5: see all options
  7. ⌘+B, ⌘+I, and ⌘+U on docs: bold, italic, underline
  8. ⌘+⎇+esc: force quit
  9. ⌘+T on an application: new tab
  10. / on google: jump to search box
  11. ⌘+⇧+T on google: duplicate tab
  12. ⌘+H on an application: hide application
  13. ⌘+Z and ⌘+⇧+Z: undo and redo
  14. ⌘+N on an application: new window
  15. ⌘+▲ on finder: go to enclosing folder
  16. ⌘+M on window: minimise
